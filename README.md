# AI Agent Workbench 2026

This repository is a portfolio of real-world AI agents designed to demonstrate practical skills in:

- Multi-agent systems
- LLM orchestration
- Tool use (web, email, automation)
- Human-in-the-loop AI systems
- Production-grade Python architecture

## Projects

1. Food Agent 🍲  
   Ingredient-based recipe discovery with web search + ranking.

2. Wedding Planner Agent 💍  
   Multi-agent system for planning events (venue, budget, schedule).

3. Email Assistant 📧  
   AI-powered inbox assistant with approval-based response system.

4. Job Search Agent 💼  
   Smart job discovery, CV matching, and application assistant.

5. clinical-agent 🧠  
   Multi-criteria decision support system using LLM reasoning + web research.

## Architecture

All agents share a common core:

- LLM layer (OpenAI API)
- Tooling system (search, email, scraping)
- Memory module
- Orchestration layer (LangGraph-based workflows)

## Goal

This project is built to demonstrate real-world AI engineering capabilities for job applications in 2026.
